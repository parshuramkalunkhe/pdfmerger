# Pdf-Editor

## How to Downlaod

- clone the repo 
```
git clone https://github.com/parshuramkalunkhe/Pdf-Editor.git
```
- install requirements
```
pip install -r requirements.txt
```
- then run the script
```
python pdfeditor.py
```

## How to use this
- Enter Path to pdf 1
- Enter Path to pdf 2 
- Enter New Merge pdf Name
